・制作期間 : 2019/7~2019/9 約３ヶ月・メンバー : エンジニア1人 営業1人 デザイナー1人
・担当 : PL,見積,設計.開発.デバッグ,リリース
・使用ライブラリ : Unity / Vuforia / Dotween /AVpro
・端末 : iOS
概要：日東電工のショールームアプリ　 機能が2つあります。 ①平置きのタッチディスプレイから課題点を選択して、映像を見終わると壁面のプロジェクションマッピングに移行する機能 ②壁面をタッチすると動画が再生される機能


・主要制作機能


・海の色 Shader
・日光 : Shader
・泡 : パーティクル
・はじめのタイトルの揺らめき演出 : VertexShader
・手紙の揺れる動き : VertexShader
・手紙のゆっくりした動き : Dotween , C#
・手紙の色 : Shader
・手紙の復元無限ループ : C#


・動画再生機能 : AVPro , C#
・クリックした後真ん中に行く動き Dotween,C#



・UDP通信＆シリアル通信 :　C#
　ディスプレイと壁面のPCはUDP通信　壁面のタッチセンサーと壁面PCは文字列をバイト変換して
　非同期待機でのシリアル通信を実装してます
デバッグ用の画面とシステムも作成
・Winsow2面出し :　bat
　batファイルで行ってます
