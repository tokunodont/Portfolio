擬似VR不動産内見アプリ
[三菱地所ハウスネット]


・制作期間 : 2020/3~2020/7 5ヶ月
・メンバー : エンジニア3人 営業1人 デザイナー1人
・担当 : PL,見積,設計.開発.デバッグ,設置,リリース
・クライアント：三菱地所ハウスネット
・使用ライブラリ : Unity / Dotween / UniRX / UniTask
・端末 : iOS
・概要：白い箱の中にPJ(プロジェクター)を5面投影し、ipadでPJを操作して擬似VRのような不動産内見ができるコンテンツ


・主要制作機能

＃動画はテスト段階やダミー画像のものです。

・検索ロジック :　C#
　 タグとフリーワード を合わせて検索可能です。
・物件生成ロジック : C#
　お客さんの方でつどコンテンツを追加していきたいとの要望だったので、
　サーバーを見て動的にボタンの数、イベント、名前等が表示される。スワイプもできる

・サーバー構築と連携 : C# , PHP , Apache ,CakePHP
　レンタルサーバーで不動産物件の数や画像を管理。お客さんが自分で入れられるようになっている。
　C#でダウンロードする。

・UDP通信 : C# , Wireshark

ipad ⇨ WindowsDeskTop ⇨ hub ⇨ プロジェクター とUDPで信号が渡っている。
写真は、UDP通信の設定とデバッグをしている所です。使用ツールは Wiresharkです。
