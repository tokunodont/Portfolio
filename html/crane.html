クレーン操者体験ゲーム
[神戸海洋博物館]


・制作期間 : 2019/10 ~ 2019/1　4ヶ月
・メンバー : エンジニア1人 営業1人 デザイナー1人 モデラー1人
・担当 : PL,見積,設計,開発,デバッグ,設置,リリース
・使用ライブラリ : Unity /カモメの群生のアセット/海アセット（unityデフォルト）UniRX,UniTask
・端末 : Windows
・概要 : 神戸海洋博物館に納品された子供の職業体験用アプリ

・主要制作機能

・クレーン操作機能
　C#とCharacterControllerで制御しています。
・当たり判定機能 :C#
　操作性上コライダーが使えないので、複数のOnTriggerEnterで処理をしています。
・２画面調整機能 : bat
　本番のディスプレイ２枚が大きさが異なるものだったためRenderTexutre２枚で調整しています。
・全体のUIアニメーション : DoTween(C#)
　DoTween.Squenceでほぼ作成してます。マリオカートなど子供向けゲーム多数のUIを参考にしました。
・結果発表の光 Shader
・目的のオブジェクトの光　Shader+PostProcessStackV2
・Timer :C#

・ユーザーのルートの記録　C#
　黒のTextureにサイドに設置したcameraから見たクレーンの移動を書きこんでます。
　軽い処理だったのでC#で済ませました。
・結果発表の光 Shader
